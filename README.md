# 500 Python Yechimlari - RoboContest

Ushbu repozitoriyda RoboContest platformasidagi 500 ta dasturlash masalalarining Python tilida yozilgan yechimlari jamlangan.

## Haqida

Ushbu loyihaning asosiy maqsadi algoritmik masalalarni yechish orqali Python dasturlash bo'yicha bilim va ko'nikmalarni rivojlantirishdir.

Barcha yechimlar masala raqamlari bo'yicha tartiblangan va sodda, tushunarli Python kodlari orqali yozilgan.

## Platforma

Masalalar quyidagi platformadan olingan:

- RoboContest — https://robocontest.uz/

## O'rganilgan mavzular

Ushbu yechimlarda quyidagi mavzular qamrab olingan:

- Python asoslari
- O'zgaruvchilar va ma'lumot turlari
- Shart operatorlari (if/else)
- Takrorlash operatorlari (for, while)
- Funksiyalar
- Ro'yxatlar (List) va massivlar
- Satrlar (String)
- Matematik amallar
- Algoritmlar

- Qidiruv algoritmlari
